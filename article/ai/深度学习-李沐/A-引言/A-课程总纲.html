                              # A-课程总纲

                              A-课程总纲

                              # No.1课程安排

                              # 一、目标

                              • B站的课程：深度学习经典和最新的模型

                              • 比如：最简单的，最老的LeNet在80年代末期提出来的模型

                              • 计算机视觉比较流行的Resnet

                              • 一个经典的一个持续模型LSTM到最近比较流行的BERT;

                              • 学深度学习的时候;离不开机器学习；机器学习很多基础知识是我们需要的

                              • 损失函数

                              • 过拟合

                              • 优化

                              • 原理

                              • 数学上的表示

                              • 代码实现

                              • 真实数据上体验一下所有这些算法的效果

                              • 不断的去尝试修改代码;以及来得到一些自己的经验

                              # 二、内容

                              • 内容包括了那么几大块：

                              • 第一块

                              • 深度学习的基础，不需要知道有机器学习;或者深度学习的背景;

                              • 第二块

                              • 深度学习的核心之一;就是卷积神经网络；从最早的开始;然后到引起深度学习这个狂潮的就是Alexnet；然后到VGG Inception;到Resnet则是最新的一些模型;

                              • 第三块

                              • 跟卷积是一个空间的神经网络;循环神经网络是一个实践上的神经网络；从最早RNN开始到新的sequence to sequence;

                              • 第四块

                              • 在两大卷积和循环神经网络之后;注意力机制就是attention;是在最近几年新出来的一个;新的神经网络的种类；

                              • 很快在自然语言处理;甚至是在NLP处理以外的;领域也得到非常广泛的应用;所以接下来会;着重介绍一下注意力机制;

                              • 第五块

                              • 机器学习的基础;就是优化算法;给一个模型怎么样得到你的网络

                              • 第六块

                              • 因为深度学习计算量比较大;需要很多的机器来跑;就是大力出奇迹的一个领域;所以会介绍一下;怎么进行高性能的计算;包括了怎么并行;怎么多GPU以及分布式;

                              • 第七块

                              • 最后会介绍两大领域;一是计算机视觉;就是包括了目标检测、语义分割;

                              • 另外一块是NLP处理;这目前是不管是深度学习也好;人工智能也好;这是最大的两大领域;这是我们整个的内容;

                              # 三、 学到

                              • 会学到什么？

                              • 一个是what

                              • what就是说有哪些技术？;知道哪个名词，做法：可以上网进行搜素教程；深度学需要哪一些技术;以及你在解决;实际问题的时候;这些哪些技术可以用来帮你解决问题

                              • 一个是how

                              • 怎么样做这个事情;当你知道有什么技术可用的时候;怎么样给一个数据;我怎么来实现这个模型;怎么进行调参得到我要的东西啊;要的精度也好好的；然后真的能够把它做出来;然后变成一个产品;或者是说变成一个paper;

                              • 一个是why

                              • 在what how下面就是一个why;为什么会这样;这是一件很难的事情;就是我们尝试来去解释;为什么这个东西很好;为什么这个东西不行;这个东西很多时候是一些直觉。

                              • 深度学习;就是大家都觉得有点像老中医;就是我知道这个东西很厉害;特别有效;但是你也很难说得清楚它到底为什么;所以这一块的话;为什么我们会给大家一些解释;直觉上的解释和数学上的解释;以及说一些我们的猜想;大家可以去自己有自己的体会;因为每个人说的不一定是对的;大家都是一个自己的理解;

                              • 这三块其实会有不同人;会有不同的需求;AI的相关从业人员;比如说产品经理;或者是说想了解一下AI做什么的话;大概知道深度学习有哪些技术;就是一些名词;知道这个名词能干什么;

                              • 然后数据科学家工程师;你要知道怎么样把你的东西做出来;就是你要手要快你最后能做出活;

                              • 那对于研究员是researcher;你除了知道有什么和怎么样做的话;你要解释为什么;这个是说你学到更深的东西;以及你能做出新的突破;你都要知道去了解为什么;特别是你得自己去想去思考为什么;

                              # No.2 深度学习介绍

                              # 一、AI地图

                              • 什么是深度学习?

                              • 先画一个很简单的人工智能的地图，

                              • X轴：模式是最早的是叫符号学; 然后接下来是概率模型; 就是基于很多统计模型; 接下来是机器学习;

                              • Y轴是说想做的东西; 最底下面是感知; 就是我得了解的是什么东西; 然后做推理; 然后能形成自己的知识; 最后做规划;

                              • 就是说最底层的就是感知; 就是我能够看到这个物体; 做推理是说我能既有看到东西; 去想象一下; 可能未来会发生什么事情; 第三个就是比较难的; 是说得根据我看到的数据; 我看到的现下来形成我自己的知识;第四个是说我能够知道所有东西的话; 我能进行一项比较长远一点的规划; 就是说我能未来怎么做; 可以做这个事情;

                              • 对领域进行简单规划

                              • 第一是自然语言处理; 在过去取得很大的进展; 但实际上还是停留在比较简单的感知上面; 比如说; 在语言处理用的比较多的是机器翻译; 机器翻译其实对一个人来讲; 比如说中文翻英文; 对一个人来说其实就是几秒钟的事情; 其实这是一个很多时候是人潜意识里面一个感知的问题; 所以说可认为说人;任何能够;几秒钟之内能够反应过来的东西; 它都属于感知的范围;

                              • 计算机视觉稍微能够往前走一点点; 就是说能够在一个图片里面; 大概可以帮你做一些推理; 然后啊;当自然语言处理最早是符号学; 因为语言是一个符号首先说; 然后当然是说; 接下来你可以用概率模型; 你也可以用机器学习; 计算机视觉呢; 因为图片里面都是一些像素; 像素的话; 啊就是你就很难用符号学来解释; 所以计算机视觉大部分是用概率模型; 或者机器学习;

                              • 深度学习它是机器学习的一种;能够做计算机视觉; 能做自然语言的处理; 它还能做比如说reforcement learning; 它还能做比较更高深一点的东西; 过去8年里面就是从从Alexnet出现开始; 最热门的方向就是说; 深度学习和计算机视觉和自然语言处理的一个结合; 这是大的方向;不管是工业界学术界; 都是关注最多的的方向;

                              # 二、图片分类

                              • 深度学习在应用图片分类上的突破
                              • 深度学习;是在做图片分类上;做了比较大的突破; Imagenet是一个比较大的图片分类数据集;它包括了1000类的自然语言;自然物体的图片; 它大概有100万张图片;
                              • 可以看到这一张图是表示说; 在过去几年里面; 时间从10年到17年; 然后你的y轴是你的错误率;
                              • 每一个点就表示一些paper的; 一些工作的他的错误率; 你可以看到10年的时候; 大家错误率还是挺高的; 最好的大概是的样子; 在10年和11年; 特别是12年; 可以看到一个比较大的一个下降; 这个就是深度学习的开始; 这可以看到在接下来5年里面啊; 深度学习模型把整个图片分类的误差降到;已经降到非常非常低了; 17年的时候; 基本上所有的团队; 他都能够做到5%以内的错误率; 基本上可以达到人类在对图片识别的一些进度了；所以我们认为是说; 在图片分类这个问题上; 机器学习已经或者说深度学习; 已经做的非常非常好了;

                              # 三、物体检测和分割

                              • ====
                              • 当你不仅仅想知道一个图片里面有什么内容; 就说我还想知道是在什么地方; 就是物体检测;
                              • 物体分割是说我想知道每一个像素; 它到底属于这个飞机还是属于这个人; 这个是一个更深层次的一个应用;

                              # 四、样式迁移

                              • ====
                              • 我有一个内容图片; 就说我拍了一个房子这是个宫殿; 然后拍了一个船; 我想把它换成一个我想匹配到这个印象派的风格; 我能把这个样式图片和内容图片; 合成一下; 就是可以大家认为就是一个滤镜; 但这个滤镜的好处是说; 我能够任意换成我想要的一个形式; 我只要找到我一张型样式的图片; 我都能把它换过去;

                              # 五、人脸合成

                              • ====
                              • 还有是人脸合成; 人脸合成是说; 这里面所有的人脸都是假的; 就都是通过机器算法; 从随机数开始的一些合成的照片; 基本可以看到; 人类是挺难分辨出来这些图片; 是真还是假了;

                              # 七、文字生成-GPT

                              • ====
                              • 文字生成模型; 是GPT three; 比如说我问一个问题; 如何来举行一个比较有效的一个董事会议; 这是我的人的问的问题; 下面这个比较长的就是第一步第二步第三; 是机器生成的模型; 基本上就是说; 语言可以真的帮你实现; 可以帮你实现给你写代码; 所以说啊; 这个也是个非常强大的语言的文字生成模型;

                              # 八、无人驾驶

                              • ====
                              • 另外一块当然是说大家可以看到媒体覆盖的比较多的就是无人驾驶; 这个是特斯拉的无人驾驶的一个图片; 可以看到左边是人坐在车上; 他在自己开车; 右边可以看到是; 在计算机用计算机视觉的模型; 把所有的车; 所有的你的规划路线你的道路跟你识别出来; 这是; 计算机视觉的一些在无人驾驶的一些应用;

                              # 九、广告点击

                              • 案例的研究
                              • 具体一个应用; 是如何利用深度学习的;以及它中间需要哪些技术; 以及哪样的人起什么样的作用; 举一个例子就是一个做广告点击; 左边这个图可以看到是在搜索框里面搜baby toy; 你会看到下面会出现很多sponsored就是你的广告; 所以我们这个任务就是说; 给你一个用户的输入; 我怎么样给;你一些广告这个问题;
                              • 这个问题可以大概; 分成三个阶段;
                              • 第一个是说当输入一个关键词的时候; 先给你找到一些相关的广告出来;
                              • 接下来就是一个机器学习的模型; 想预测看到一个广告之后; 每个人的点击会怎么样子; 这上面又写了一个p; 等于就是你一个人会点击这个广告的概率;
                              • 第三步那么就是说; 根据你的点击率; 和每一次用户点了之后; 广告主会付你多少钱; 把它相乘; 就是点击率乘以进价; 就会得到一个数字; 然后把它作为一个排序; 排名高的会出在最前面; 排名最低的可能会舍去; 这个就是一个广告点击的三个步骤; 触发广告进行预测一个点击率; 最后做一个排序;

                              • ====
                              • 最重要的就是;我要预测一个点击率; 这是一个机器学习的模型;这个有大概这个几个步骤;
                              • 第一步是说我看了一个广告的时候; 我首先要进行特征提取; 就是比如说包括了广告主; 是谁在打广告; 这个广告是什么样的描述信息;一些产品的图片; 然后把这些特征放进一个模型之后; 我就可以预测他一个点击的预测;
                              • 这是一个很简单的机器学习; 它就是这么两个步骤; 特征提取+模型的预测; 然后我们可以把所有过去的广告展现;和用户的点击存下来；
                                比如说过去三个月里面;所有看过的广告; 我把它存在一个地方; 我们把它; 然后因为我知道用户点还是没有点; 所以是有真实的用户点击行为; 把它作为一个训练数据; 把这个数据大概比如说100个g; 或者一个t; 然后把它做特征提取; 然后放在一个模型里面进行训练; 这是我的训练模块;
